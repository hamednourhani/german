# Essential verbs for B1 — conjugation reference (Present + Perfekt)

This file lists common B1 verbs with:
- short English translation
- Present tense conjugation for all persons (ich / du / er-sie-es / wir / ihr / sie-Sie)
- Perfekt conjugation (auxiliary + past participle) for all persons

Note: For movement verbs some contexts use **sein** as auxiliary (marked). Regular verbs form Perfekt with **ge-** + stem + **-t**; many common strong/irregular verbs have irregular participles.

Format: Infinitive — English
Present: ich, du, er/sie/es, wir, ihr, sie/Sie
Perfekt: ich ..., du ..., er ..., wir ..., ihr ..., sie/Sie ...

1) sein — to be
Present: ich bin, du bist, er ist, wir sind, ihr seid, sie sind
Perfekt: ich bin gewesen, du bist gewesen, er ist gewesen, wir sind gewesen, ihr seid gewesen, sie sind gewesen
